# Simple Interest Calculator

## Description
This project is a Bash script that calculates simple interest based on the provided inputs.

## Inputs
The script requires the following inputs:
- **Principal**: The initial amount of money
- **Rate of Interest**: The interest rate per annum (%)
- **Time Period**: The time duration in years

## Formula
The simple interest is calculated using the formula:

```
Simple Interest = (Principal × Rate × Time) / 100
```

## Example
To run the script:

```bash
bash simple-interest.sh
```

Then follow the prompts to enter:
1. Principal amount
2. Rate of interest
3. Time period

The script will output the calculated simple interest.
